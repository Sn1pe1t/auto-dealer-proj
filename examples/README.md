# Примеры использования autodealer_core

## basic_usage.py

Базовый пример использования основных функций библиотеки:

- Инициализация БД
- Получение информации об автомобиле
- Чтение списка продаж
- Просмотр деталей конкретной продажи
- Генерация отчета по продажам

### Запуск

```bash
python basic_usage.py
```

### Требования

```bash
pip install autodealer-core
```
